PM Pilot is an open-source toolkit that connects your PM workflow to Claude. Everything here is a markdown file. You can copy it, paste it, or install it. Nothing requires a subscription.

## What's in the box

25 skills across four categories:

- **PM Core:** the 12 skills you'll use every week: meeting prep, PRDs, weekly status, market sizing, prioritization, research synthesis
- **Productivity:** session management, handoffs, multi-agent orchestration
- **Dev:** TDD workflow, verification, security review (if you touch code)
- **Content:** market research, voice enforcement, Substack formatting

## Three ways to use it

You don't need Claude Code to start. Pick the setup that fits where you are:

| Setup | Time | What you get |
|---|---|---|
| Copy-paste | 0 min | Works in ChatGPT, Gemini, or claude.ai today |
| Claude Desktop | 5 min | Skills persist between sessions |
| Claude Code CLI | 20 min | Live Jira, Slack, Calendar, full power |

Start with [Getting Started](/pm-pilot/getting-started) or jump straight to a skill.

## Start here

If you're new: read [Getting Started](/pm-pilot/getting-started) first.

If you're a PM who wants to save time today: go to [/meeting-prep](/pm-pilot/skills/meeting-prep). It's the most immediately useful skill.

If you want to write PRDs faster: go to [/prd](/pm-pilot/skills/prd).

If you're setting up weekly reporting: go to [/weekly-status](/pm-pilot/skills/weekly-status).
