## Why this exists

Claude Code is powerful. The docs are thorough. But if you're not already comfortable in the terminal, both feel like they were written for someone else.

This guide bridges that gap.

We wrote it for PMs, designers, founders, and "non-technical" people who keep hearing about AI coding tools and want in, without pretending they need a CS degree first.

## Quick start

You don't need to clone this repo to use the guide. Just visit **[claudecodeguide.dev](https://claudecodeguide.dev)**.

If you want to run it locally or contribute:

```bash
# Clone the repo
git clone https://github.com/mshadmanrahman/claudecode-guide.git
cd claudecode-guide

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [localhost:3000](http://localhost:3000) and you're in.

## Contributing

Found a typo? Have a better explanation? Want to add a template?

1. Fork the repo
2. Create a branch (`git checkout -b improve/something`)
3. Make your changes in `content/docs/`
4. Open a PR

Every article is an `.mdx` file in `content/docs/`. You don't need to know React. If you can write Markdown, you can contribute.
